# 批量依赖

    pip freeze > requirements.txt
    查看源文件
    
    pip的freeze命令用于生成将当前项目的pip类库列表生成 requirements.txt 文件：
    
    如果要安装requirements.txt中的类库内容，那么你可以执行pip install -r requirements.txt.
