

bin 目录下按系统信息选择目录执行

```
bin/
├── jsw-license
│   └── LICENSE.txt
├── linux-x86-32
│   ├── lib
│   │   └── libwrapper.so
│   ├── sonar.sh
│   └── wrapper
├── linux-x86-64
│   ├── lib
│   │   └── libwrapper.so
│   ├── SonarQube.pid
│   ├── sonar.sh
│   └── wrapper
├── macosx-universal-64
│   ├── lib
│   │   └── libwrapper.jnilib
│   ├── sonar.sh
│   └── wrapper
├── windows-x86-32
│   ├── InstallNTService.bat
│   ├── lib
│   │   └── wrapper.dll
│   ├── StartNTService.bat
│   ├── StartSonar.bat
│   ├── StopNTService.bat
│   ├── UninstallNTService.bat
│   └── wrapper.exe
└── windows-x86-64
    ├── InstallNTService.bat
    ├── lib
    │   └── wrapper.dll
    ├── StartNTService.bat
    ├── StartSonar.bat
    ├── StopNTService.bat
    ├── UninstallNTService.bat
    └── wrapper.exe

```

```
sonar.sh start/stop/status
```
